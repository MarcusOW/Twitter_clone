# 🐦 Twitter Clone

Um clone do Twitter/X desenvolvido com Django (backend) e React (frontend), utilizando autenticação JWT, API RESTful e banco de dados PostgreSQL.

---

## 🚀 Tecnologias

### Backend

- Django 6.0
- Django REST Framework
- Django REST Framework SimpleJWT (autenticação JWT)
- PostgreSQL (com suporte a SQLite para desenvolvimento)
- WhiteNoise (servir arquivos estáticos)
- Gunicorn (servidor WSGI)
- Poetry (gerenciador de dependências)

### Frontend

- React 18
- Vite
- React Router DOM
- Axios
- Tailwind CSS

### DevOps

- Docker & Docker Compose
- GitHub Actions (CI)
- Render (deploy)

---

## 🛠️ Como rodar o projeto localmente

### Pré-requisitos

- Python 3.14+
- Node.js 20+
- Poetry
- PostgreSQL (ou SQLite para testes rápidos)

### 1. Clone o repositório

```bash
git clone https://github.com/SEU_USUARIO/twitter-clone.git
cd twitter-clone
```
